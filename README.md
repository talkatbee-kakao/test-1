## 텍스트 꾸미기

- **굵게**: 굵게 나타낼 텍스트의 앞뒤를 **또는 __로 감쌉니다.
- **기울임체**:기울임체로 나타낼 텍스트의 앞뒤를 *또는 _로 감쌉니다.
- **굵음 기울임체**:굵은 기울임체로 나타낼 텍스트의 앞뒤를 ***또는 ___로 감쌉니다.
- **cnlthtjs**:취소선을 나타낼 텍스트의 앞뒤를 ~로 감쌉니다.
